European Space Agency VHDL Repository
==========================================

This repository is intended for storing public information related to VHDL (VHSIC Hardware Description Language), such as VHDL models, VHDL code examples, documents related to VHDL etc.

This repository is maintained by the Spacecraft Control and Data Systems Division, WS, at ESTEC (European Space Research and Technology Centre).

Contact points are:

P. Sinander, e-mail psi@ws.estec.esa.nl
